# FEP Semester Project

This repository will contain the codebase for an integrated autonomous system utilizing a Jetson Nano, 5G connectivity, and computer vision.

---

## 📂 Targeted Repository Structure

* **`/mavlink-communication`**
    * Core code managing data exchange between the **Jetson Nano** and **Pixhawk** flight controller using the MAVLink protocol.
* **`/5g-module`**
    * Scripts, configuration, and network drivers related to the **5G hardware module** for high-speed telemetry and data streaming.
* **`/computer-vision`**
    * Image processing, object detection, and perception algorithms running locally on the Jetson Nano.
